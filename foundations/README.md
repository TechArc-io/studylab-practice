# Foundations practice

Open `index.html` directly in a browser for the early HTML, CSS, and JavaScript exercises. This area deliberately avoids React and build tooling so you can see what the browser receives.

- Improve semantics and keyboard access in `index.html`.
- Build responsive layout rules in `styles.css`.
- Add small, observable interactions in `app.js`.

Keep the page usable without JavaScript and test at narrow and wide widths.
